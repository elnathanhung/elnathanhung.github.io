Together We Run

An Interactive Running Game

by Off The Beaten Track x Kult

Together We Run is an interactive running game console commissioned by Sport Singapore as part of their 2024 lifestyle campaign “Together We Play.” This campaign aims to foster healthy, active lifestyles among the public. Together We Run immerses players in a third-person running game where the objective is to collect as many coins as possible and take as many steps as they can within a 60-second time limit. The game features iconic Singaporean landmarks, such as the Marina Bay Sands towers and Housing Development Board-like buildings, enriching the gameplay with a local touch.

The game was developed and programmed in Unity. Player running motion is captured using force-sensitive resistor (FSR) sensors embedded under a carpet. These sensors detect the analog values of the player's steps, which are then transmitted to an Arduino microcontroller. The Arduino processes these values to determine if the Unity avatar should start or stop running. By calculating the average difference in analog values from one moment to the next, the system accurately senses jogging in place, enabling seamless gameplay.

Together We Run includes an onboarding sequence to familiarize players with the controls and objectives. Following the onboarding, a 60-second countdown begins, during which the player must run to collect coins and take as many steps as possible. The system tracks the number of steps taken and coins collected within this time frame, offering a fun and engaging way to encourage physical activity.
