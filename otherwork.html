"Quarantine Order Isn’t A Holiday" is a published Meta (Facebook) Story filter that prototypes a game that references Singapore’s Quarantine Order (QO) within 2019/2020, when the COVID-19 pandemic was still fresh. The game comedically references the idea of having to be sent to a QO site (typically a luxurious hotel) when tested positive for the virus, while promoting the notion of going out of one’s way to catch the virus. The game is a simple avoid target game, however, whether catching the virus is losing or winning the game is left ambiguous. The game prototype was built with the Meta software, SparkAR.

"Explorer" is a prototype travel mobile application catered for young Singaporeans who need travel and activity inspiration. Within the app, you can experience the functionality of sharing inspiration through a social media section, the usability of an in-software shared planner, including the convenience of an e-commerce/booking platform. The app is an all-in-one system for travelers who wish to explore activities, lodging, air tickets, and F&B outlets for both local and abroad. Within the short video walkthrough, you will see a flow of the e-commerce section that demonstrates the UI and different functions of the details pages. The walkthrough is a glimpse of the seamless and minimal user experience behind the app design.

The goal of the work was to remake the Graffiti Research Lab laser graffiti project. I used TouchDesigner to recreate the graffiti through blob tracking and projection. The colours of the strokes are defined by a “noise operator”, resulting in a vibrant visual effect. Opencv allows us to map the webcam image input according to the projection output. The input image is distorted to fit the dimensions of the projection using a “stoner palette”.

"The Day Before" is a 'choose-your-own-adventure' narrative inspired by the film "Primer" (2004). This game was created with TouchDesigner using containers and buttons which form a basic user interface with selected paths. Imagine you are a broke inventor who is in desperate to win the lottery. Your only way to win is to lose first, travel back to the day before and submit the winning numbers. Things start to go wrong.

"The Terrain Project" is a visual sound-waves that response to audio signals done with TouchDesigner. These signals can be sourced from either an external microphone or instrument, which provides live input signals (or “Live In”), or signals from an internally stored playlist, mp3 tracks, or movie files that are saved within the project directory. The signal to be channeled into the programming environment is analyzed as 3 different patterns: high, middle, and low frequencies. These are translated into waves which are visualized as columns in a grid. The result is a moving three-dimensional oscilloscope (drawn in x, y, z axes) containing wave-like qualities.
